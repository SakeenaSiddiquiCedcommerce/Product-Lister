== Description ==

Product Lister for Etsy allows user to sync their WooCommerce store and its products to the Etsy Marketplace.

A few notes about the sections above:

*   Upload Simple Products to Etsy Marketplace
*   Upload Variable Products to Etsy Marketplace
*   Auto Sync Price and Inventory

= Main Features of Free Version  =

* Category mapping with etsy category.
* Create and edit profile settings.
* Easy and simple configuration.
* Easy to use and install.
* Easy to configure.
* Sync upto 100 products from woocommerce to etsy.

= Main Features of Premium Version =

* Sync unlimited products and orders.
* Auto upload products feature from Woocommerce to Etsy
* Auto import products feature from Etsy to Woocommerce [ Import Module ].
* Shipment Tracking plugin compatibility [ will update tracking information automatically to Etsy from Woocommerce ].
* Support digital files upload from Woocommerce to Etsy
* Auto product inventory synchronization.
* Auto Order Fetching from the etsy store to your woocommerce panel.
* Easy and simple configuration.
* Easy to use and install.
* Easy to configure.


You can get premium version from [here](https://woocommerce.com/products/etsy-integration-for-woocommerce/)

== Installation ==

= Automatic installation =

In the search field type " Product Lister for Etsy" and click Search Plugins. Once you've found our  'Product Lister for Etsy' you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

No Automatic updates should work in free version ,you can get automatic update in premium version .
